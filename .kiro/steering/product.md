# Product Overview

A full-stack to-do list web application with user authentication and cloud storage. Users can securely manage their tasks from any device with real-time synchronization powered by Supabase.

## Key Features

- User authentication (sign up/sign in with email)
- Add, edit, delete, and mark tasks as complete
- Real-time updates across devices and sessions
- Secure cloud storage with PostgreSQL
- Row Level Security ensures data privacy
- Responsive design for mobile and desktop
- Inline task editing
- Persistent storage across devices
